Guide
Comment utiliser le Rack
📱 Étape 1 : Scannez le QR Code

Où est le QR ? Il est fixé sur le montant du RACK ATTENTE.

Comment faire :

1️⃣ Ouvrez l'appareil photo de votre téléphone

2️⃣ Pointez vers le carré noir/blanc du QR code

3️⃣ Une notification apparaît → cliquez dessus

4️⃣ Vous arrivez sur cette application ! ✅

📍 Étape 2 : Choisissez votre emplacement

Vous voyez le RACK entier avec 24 emplacements :

🟡 Haut (D) → D1 à D6

🔵 Milieu-haut (C) → C1 à C6

🟢 Milieu (B) → B1 à B6

🟠 Bas (A) → A1 à A6

Action : Cliquez sur l'emplacement où vous mettez vos colis

✏️ Étape 3 : Ajoutez vos colis

🔴 N° Commande / Référence (obligatoire par colis)

Exemple : CMD001, REF12345…

🟠 Numéros AFS

Plusieurs AFS par colis ? → Ajoutez-les un par un

➕ Plusieurs colis sur le même emplacement ?

Ajoutez autant de colis que nécessaire sur un même emplacement !

📊 Pour B4, B5, B6 :

Indiquez s'il reste de la place !

🎉 Vous maîtrisez le RACK !

📱 1. Scanner le QR code au rack

📍 2. Cliquer sur votre emplacement

✏️ 3. Ajouter un ou plusieurs colis

✓ 4. Valider chaque colis

❓ 5. Cliquer sur 🤖 pour poser une question

📦 Sortir un colis ? Cliquer sur ✕ à côté du colis.

⚠️ B4, B5, B6 : pensez à indiquer s'il reste de la place !
