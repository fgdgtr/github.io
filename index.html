Guide
Comment utiliser le Rack
📱 Étape 1 : Scannez le QR Code

Où est le QR ? Il est fixé sur le montant du RACK ATTENTE.

Comment faire :

1️⃣ Ouvrez l'appareil photo de votre téléphone

2️⃣ Pointez vers le carré noir/blanc du QR code

3️⃣ Une notification apparaît → cliquez dessus

4️⃣ Vous arrivez sur cette application ! ✅

📍 Étape 2 : Choisissez votre emplacement

Vous voyez le RACK entier avec 24 emplacements :

🟡 Haut (D) → D1, D2, D3, D4, D5, D6

🔵 Milieu-haut (C) → C1, C2, C3, C4, C5, C6

🟢 Milieu (B) → B1, B2, B3, B4, B5, B6

🟠 Bas (A) → A1, A2, A3, A4, A5, A6

Action : Cliquez sur l'emplacement où vous mettez votre brut

💡 Exemple : vous mettez sur B4 ? → Cliquez sur "B4"

✏️ Étape 3 : Remplissez les infos du brut

🔴 N° Commande / Référence (obligatoire !)

Exemple : CMD001, REF12345, N° client, etc.

🟠 Numéros AFS (optionnel)

Plusieurs ? → Séparez par des virgules ou espaces

📊 Délai max : 90 jours

L'app calcule automatiquement :

🟢 0-15 jours → Vert

🔵 15-21 jours → Bleu

🟡 21-28 jours → Jaune

🔴 +28 jours → Rouge (attention !)

✓ Étape 4 : Validez l'entrée

Vous avez rempli la référence ? Maintenant :

🔘 Cliquez sur "✓ Valider l'entrée"

✅ Succès ! Votre brut apparaît immédiatement sur le rack

📡 IMPORTANT : Tous les autres opérateurs voient la modification en temps réel

Aucun doublon, tout est synchronisé automatiquement

🎉 Vous maîtrisez le RACK !

Résumé rapide :

📱 1. Scanner le QR code au rack

📍 2. Cliquer sur votre emplacement (A1, B4, D6, etc.)

✏️ 3. Entrer la N° Commande + AFS (optionnel)

✓ 4. Cliquer "Valider l'entrée"

💡 Astuce importante :

La couleur change AUTOMATIQUEMENT selon les jours passés :

Plus ça reste longtemps → Plus ça devient rouge 🔴

Si c'est rouge = URGENCE ! Traiter en priorité
