Guide
Comment utiliser le Rack
📱 Étape 1 : Scannez le QR Code

Où est le QR ? Il est fixé sur le montant du RACK ATTENTE.

Comment faire :

1️⃣ Ouvrez l'appareil photo de votre téléphone

2️⃣ Pointez vers le carré noir/blanc du QR code

3️⃣ Une notification apparaît → cliquez dessus

4️⃣ Vous arrivez sur cette application ! ✅

📍 Étape 2 : Choisissez votre emplacement

Vous voyez le RACK entier avec 24 emplacements :

🟡 Haut (D) → D1 à D6

🔵 Milieu-haut (C) → C1 à C6

🟢 Milieu (B) → B1 à B6

🟠 Bas (A) → A1 à A6

Action : Cliquez sur l'emplacement où vous mettez votre palette

✏️ Étape 3 : Remplissez les infos

🔴 N° Commande / Référence (obligatoire)

Exemple : CMD001, REF12345…

🟠 Numéros AFS (optionnel)

Plusieurs ? → Séparez par virgules ou espaces

📊 Pour B4, B5, B6 :

Indiquez s'il reste de la place sur la palette !

✅ Oui · ⚠️ Un peu · ❌ Non

🔒 Les emplacements réservés

Certains emplacements sont réservés à une entreprise.

Vous les reconnaissez au symbole 🔒 affiché sur la case.

Que faire si vous avez une palette pour cette entreprise ?

1️⃣ Cliquez sur l'emplacement réservé

2️⃣ Entrez le N° commande et les AFS

3️⃣ Cliquez "✓ Valider l'entrée"

✅ La réservation reste en place — l'emplacement est juste occupé

🎉 Vous maîtrisez le RACK !

📱 1. Scanner le QR code au rack

📍 2. Cliquer sur votre emplacement

✏️ 3. Entrer la N° Commande + AFS

✓ 4. Valider

🔒 Emplacement réservé ? Entrez quand même le colis directement dessus, la réservation reste.

⚠️ B4, B5, B6 : pensez à indiquer s'il reste de la place !
